Pipeline Utilities

  • Pipeline utilities is a small C++ utility, written in C++17. This utility is mainly for filtration from data structures, can be used in both C++ standard libraries like std::list , std::vector , and xperimental::list_sm , xperimental::array_sm . It's fast and concise, increases industrial productivity.

    Objectives are:

  • Concise and yet readable
  • Memory management
  • Ability to handle Inter-process Commmunications
  • Productivity

    • Supported platform

  • Win32, GCC-7.5.0 (MinGW)
  • Linux, GCC-7.5.0

    • Pipeline functions:

  • xperimental::from
  • xperimental::filter
