Xperimental Library

  • Xperimental Library is a collection of small cross-platform C++ libraries and utilities written in C++17- powerfull and easy to implement. Some of the libraries are datastructures as in C++/stl. Instead of using heap memory, used shared-memory that enhancing inter-communication among different precesses

    • Supported platform

  • Win32, GCC-7.5.0 (MinGW)
  • Linux, GCC-7.5.0
